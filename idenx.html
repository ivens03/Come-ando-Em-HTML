História Dos Computadores

O computador é um conjunto de componentes eletrônicos capaz de executar variados tipos de algoritmos e tratamento de informações. Este dispositivo pode possuir inúmeros atributos, dentre eles: armazenamento de dados, processamento de dados, cálculo em grande escala, desenho industrial, tratamento de imagens gráficas, realidade virtual, entretenimento e cultura.

História Do HTML

A sua evolução começou em 1989, quando Tim Berners-Lee, um cientista da computação britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear), desenvolveu a ideia da World Wide Web. A primeira versão do HTML, conhecida como HTML 1.0, foi lançada em 1991.

Códigos Básicos De HTML

A tag HTML

Na estrutura do nosso documento, antes de começar a colocar o conteúdo.

A tag head

A tag head contém informações sobre o documento HTML que são de interesse somente do navegador e para outros serviços da web.

A tag body

A tag body contém o corpo de um documento HTML, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site.

A instrução DOCTYPE

O DOCTYPE não é uma tag HTML, mas uma instrução especial. Ela indica para o navegador qual versão do HTML deve ser utilizada para exibir a página.
